# TIGEST
Sistema de Gestão de Recursos de TI, Construido usando Windows Form em C#

## Resumo
Sistema Construido utilizando conceitos e recursos do Windows Form, seu objetivo é ser uma plataforma de gestão de recursos de departamento.

Divido em quatro partes, o TIGEST possui uma gama de ferramentas que auxiliam a gestão e trazem ao usuário final a facilidade e interatividade de modo que não é necessário um conhecimento técnico para realização do processo. Sua restrição está relacionada aos processos internos de um departamento, sendo necessário a adaptação conforme necessidades.
